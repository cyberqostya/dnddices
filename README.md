## О приложении

DnD Dices - это мобильный набор виртуальных кубиков для настольных ролевых игр. Приложение можно открыть как обычный сайт или запустить как Telegram Mini App, чтобы быстро собрать нужный пул кубиков и бросить его прямо с телефона.

### Возможности для игроков

- Добавление разных типов кубиков: d2, d4, d6, d8, d10, d12, d20 и d100.
- Одновременный бросок всех выбранных кубиков с анимацией и звуком.
- Автоматический подсчет общей суммы броска.
- Отображение результата на каждом отдельном кубике.
- Подсветка критических значений: удачный максимум и неудачная единица.
- Режим `KARMA`, который мягко балансирует серии слишком хороших или слишком плохих бросков.
- Режим `REROLL` для переброса отдельных кубиков без сброса всего пула.
- Режим `EDIT` для удаления лишних кубиков из набора.
- Сброс всей башни кубиков одной кнопкой.
- Звуковые эффекты бросков, монетки и переключения кармы.
- Тактильная отдача в Telegram Mini App: кнопки и броски кубиков ощущаются через вибрацию.
- Адаптированный интерфейс под мобильный экран и Telegram WebApp.

### Что реализовано технически

- Проект написан на чистом HTML, CSS и JavaScript без фреймворков и сборщика.
- Логика разделена на ES-модули и классы: кубики, кнопки настроек, общий результат, Telegram API и аудио.
- Кубики реализованы как независимые экземпляры класса `Dice` со своим DOM-узлом, состоянием, анимацией, звуком и результатом.
- Для звука используется Web Audio API: аудиофайлы загружаются заранее, а `AudioContext` создается и восстанавливается только после пользовательского действия.
- Добавлена обработка мобильных ограничений звука: восстановление аудиоконтекста после возврата приложения из фона.
- Интеграция с Telegram WebApp API: инициализация Mini App, Telegram-only стили, отключение вертикальных свайпов и Haptic Feedback.
- Интерфейс использует CSS-анимации для бросков, тряски в режиме редактирования, появления результата и декоративной анимации кнопок.
- Реализована простая PWA-часть через Service Worker для кеширования ресурсов.
- Код приложения организован так, чтобы функциональность сайта и Telegram Mini App использовала одну и ту же базу.
